# Gestor de Notas de Alumnos (CLI)

¡Hola! Bienvenido a este mini-proyecto. Básicamente, es un sistema de gestión por línea de comandos (CLI) súper sencillo para registrar alumnos, cargar sus notas y sacar un par de estadísticas rápidas.

Está pensado para ser rápido, al pie de la letra, y funciona 100% en la terminal sin interfaces gráficas pesadas.

## ¿Qué hace exactamente?

- Te permite ingresar la cantidad de alumnos que quieras (¡sin límite!).
- Te pide el nombre, edad y nota de cada uno.
- Cuando terminás de cargar, te devuelve:
  1. La lista completa de todos los alumnos ingresados.
  2. Un ranking con las notas ordenadas de **mayor a menor**.
  3. El **promedio general** de todo el grupo.

## Tecnologías utilizadas

Para demostrar cómo se resuelve el mismo problema con distintas lógicas, el sistema está escrito en dos lenguajes distintos. Podés usar el que más te guste:

- **Python 3** 🐍
- **JavaScript (Node.js)** 🟨
